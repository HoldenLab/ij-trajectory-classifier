#Trajectory classifier
This plugin loads trajectories from [TrackMate](http://imagej.net/TrackMate), characterize them using [TraJ](https://github.com/thorstenwagner/TraJ) and classificate them into normal diffusion, subdiffusion, confined diffusion and directed/active motion by a random forest approach (through [Renjin](http://www.renjin.org/))
